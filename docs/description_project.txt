TelegramBot

Для пользователя:
При запуске бота он предлагает добавить выбрать ресурсы, с которых будут приходить новости в телеграм.
Пользователь отмечает необходимые ему ресурсы.
После этого бот запрашивает частоту обновления новостной ленты.
 Пользователь выбирает из предложенных вариантов. Используется RSS.
 При обновлении ленты добавляются все новые публикации со времени последнего запроса.
Отображение информации идёт через телеграм.

Для администратора:
Для администратора в веб-интерфейсе доступна статистика использования бота:
сколько аккаунтов его используют, на какие рассылки подписаны, есть доступ к базе данных с аккаунтами пользователей,
списками рассылок для них и частоте обновления ленты для каждого пользователя.
Для каждого пользователя возможно как отключить телеграм-бот,
 так и подключать/отключать ресурсы из списка для новостной ленты.
