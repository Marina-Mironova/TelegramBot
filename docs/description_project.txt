Телеграм-бот:
По запросу пользователя бот показывает погоду в выбранном месте (выбор кнопкой)
 на настоящий момент или прогноз на ближайшее время.

Базовая версия приложения:  по команде /start бот регистрирует пользователя в базе данных
с указанием даты и времени регистрации
Команда /help - объяснять, что делает бот, и для чего нужны другие команды.


Сущности: User
Свойства(поля)
User: Id чата, имя, фамилия, юзернэйм, дата и время регистрации
Методы класса: toString()




Для работы приложения ключевыми являются два момента:
1) взаимодействие приложения с базой данных
2) взаимодействие приложения со сторонним ресурсом и телеграмом,
 т.е.успешная отправка приложением погоды и получение его пользователем в приложении телеграм
